Пример

[ {"date": "2018-07-15", "match": "вап", "bet": 1, "ratio": 0.005, "score": "0-15", "result": false}, {"date": "2019-06-28", "match": "Blooming - The Strongest", "bet": 1, "ratio": 2.82, "score": "4-1", "result": true} ]

Этот пример добавляет две записи в БД. Даже если хочешь добавить одну запись - все равно ставь квадратные скобки.

Скобки [ ] - значит массив. Внутри массива через запятые перечислены элементы массива. В фигурных скобках { } - объекты - элементы массива. В объектах пары ключ - значение перечисляются через запятую. Там, где строки - обрамляй двойными кавычками, где числа и false/true - без кавычек.

Дату писать именно так, через тире, ГГГГ-ММ-ДД.
